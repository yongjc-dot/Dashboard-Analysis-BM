# Dashboard Daerah Seberang Perai Tengah  
## Fokus: P.045 Bukit Mertajam

Dashboard ini dibangunkan sebagai papan pemuka analitik setempat bagi **Daerah Seberang Perai Tengah**, dengan fokus khas kepada **P.045 Bukit Mertajam** dan tiga kawasan Dewan Undangan Negeri (DUN) di bawahnya, iaitu **N.13 Berapit**, **N.14 Machang Bubuk** dan **N.15 Padang Lalang**.

Dashboard ini bertujuan membantu pengguna memahami profil penduduk, struktur ekonomi, indikator sosial, kemudahan awam, serta aset setempat seperti landmark dan tarikan komuniti.

---

## 1. Tujuan Dashboard

Dashboard ini disediakan untuk:

- Menyediakan gambaran menyeluruh mengenai Daerah Seberang Perai Tengah.
- Membezakan dengan jelas antara **daerah pentadbiran** dan **kawasan Parlimen**.
- Menjadikan Bukit Mertajam sebagai fokus analisis dalam konteks daerah yang lebih tepat.
- Menyokong penyediaan laporan, pembentangan, perancangan setempat dan komunikasi data kepada orang awam.
- Menggabungkan data statistik rasmi dengan elemen penceritaan setempat.

---

## 2. Penjelasan Geografi

**Bukit Mertajam bukan daerah pentadbiran.** Oleh itu, dashboard ini menggunakan struktur geografi berikut:

| Tahap Geografi | Kegunaan Dalam Dashboard |
|---|---|
| Daerah Seberang Perai Tengah | Asas utama untuk MyLocal Stats, KDNK, kemudahan awam dan indikator pentadbiran |
| P.045 Bukit Mertajam | Fokus khas bagi analisis demografi dan profil kawasan Parlimen |
| N.13 Berapit | Pecahan DUN dalam P.045 Bukit Mertajam |
| N.14 Machang Bubuk | Pecahan DUN dalam P.045 Bukit Mertajam |
| N.15 Padang Lalang | Pecahan DUN dalam P.045 Bukit Mertajam |
| Pulau Pinang | Digunakan sebagai penanda aras negeri jika perlu |

Nota ini penting bagi memastikan semua statistik dibentangkan mengikut tahap geografi yang betul dan tidak mengelirukan pengguna.

---

## 3. Kandungan Dashboard

### 3.1 Ringkasan Utama
Bahagian ini memaparkan indikator utama seperti jumlah penduduk, keluasan kawasan, kepadatan penduduk, pendapatan isi rumah, kadar kemiskinan dan indikator ekonomi utama.

### 3.2 Demografi
Bahagian demografi memaparkan taburan jantina, kumpulan umur, struktur umur bekerja, komposisi penduduk dan perbandingan kawasan berkaitan jika data tersedia.

### 3.3 Ekonomi
Bahagian ekonomi memberi tumpuan kepada pendapatan isi rumah, KDNK daerah, aktiviti ekonomi utama, perkhidmatan, perdagangan, pembuatan dan potensi ekonomi setempat.

### 3.4 Sosial dan Kemudahan
Bahagian ini merangkumi indikator berkaitan pendidikan, kesihatan, keselamatan, kemudahan awam, fasiliti komuniti dan infrastruktur asas.

### 3.5 Fokus Bukit Mertajam
Bahagian ini menerangkan peranan Bukit Mertajam sebagai bandar utama dalam Daerah Seberang Perai Tengah, termasuk fungsi bandar, peranan ekonomi dan komuniti, kedudukan strategik, landmark dan identiti setempat.

### 3.6 DUN Dalam P.045 Bukit Mertajam
Bahagian ini memberi tumpuan kepada tiga DUN: N.13 Berapit, N.14 Machang Bubuk dan N.15 Padang Lalang. Bahagian ini boleh digunakan untuk analisis perbandingan dalaman kawasan Parlimen.

### 3.7 Peta dan Landmark
Dashboard turut mengandungi elemen peta dan lokasi penting seperti Bukit Mertajam town centre, St. Anne’s Church, Bukit Mertajam Recreational Forest / Cherok Tok Kun, Mengkuang Dam dan kawasan tumpuan komuniti serta ekonomi.

## 5. Nota Rujukan Masa

Setiap indikator mungkin mempunyai tahun rujukan yang berbeza. Contohnya:

- Data penduduk dan demografi mungkin merujuk kepada MyCensus 2020 atau anggaran penduduk terkini.
- Data ekonomi seperti pendapatan, kemiskinan dan KDNK mungkin mempunyai tahun rujukan yang berbeza.
- Data kemudahan awam daripada MyLocal Stats atau agensi pentadbiran mungkin bergantung kepada tahun penerbitan terkini.
- Data landmark dan lokasi bersifat deskriptif dan perlu disahkan dengan sumber pentadbiran atau geospatial jika digunakan untuk analisis rasmi.

Dashboard perlu memaparkan **sumber data, tahun rujukan dan tahap geografi** bagi setiap indikator utama.

Contoh:
> Sumber: DOSM, MyLocal Stats 2024  
> Geografi: Daerah Seberang Perai Tengah  
> Nota: Data mungkin berbeza mengikut tahun rujukan indikator.

## 6. Cara Membuka Dashboard

Dashboard ini ialah laman web statik. Untuk membuka dashboard:

1. Buka folder dashboard.
2. Cari fail `index.html`.
3. Klik dua kali pada fail tersebut.
4. Dashboard akan dibuka dalam pelayar web seperti Google Chrome, Microsoft Edge atau Firefox.

Contoh lokasi fail tempatan:

```text
E:/Penyediaan Analisis BM/Seberang Perai Tengah/index.html
```

---

## 7. Cara Menyebarkan Dashboard Secara Online

Dashboard ini boleh dimuat naik ke platform hosting statik seperti Netlify, GitHub Pages atau Vercel.

### Kaedah Netlify Drop

1. Pergi ke laman Netlify Drop.
2. Pastikan folder dashboard telah diekstrak sepenuhnya.
3. Seret folder yang mengandungi `index.html` ke Netlify Drop.
4. Netlify akan menjana pautan awam secara automatik.
5. Pautan tersebut boleh dikongsi kepada pengguna lain.

## 9. Cadangan Penambahbaikan

Penambahbaikan yang boleh dibuat pada masa hadapan:

- Menambah sambungan data langsung kepada OpenDOSM atau data.gov.my.
- Menambah peta interaktif dengan sempadan daerah, Parlimen dan DUN.
- Menambah lapisan lokasi kemudahan awam seperti sekolah, klinik, balai polis dan balai bomba.
- Menambah fungsi muat turun data dalam format CSV.
- Menambah papan pemuka perbandingan antara daerah di Pulau Pinang.
- Menambah analisis trend mengikut tahun.
- Menambah visual yang membezakan indikator rasmi, proksi daerah dan data pentadbiran.

## 10. Nota Penting Untuk Pengguna

Dashboard ini bertujuan sebagai alat analisis, visualisasi dan penceritaan data. Sebarang penggunaan untuk tujuan rasmi perlu merujuk kepada sumber asal dan tahun rujukan setiap indikator.

**Nota utama:**
> Bukit Mertajam bukan daerah pentadbiran. Oleh itu, indikator yang memerlukan tahap daerah seperti MyLocal Stats, KDNK dan kemudahan awam perlu menggunakan Daerah Seberang Perai Tengah sebagai asas analisis. Bukit Mertajam boleh dikekalkan sebagai fokus kawasan Parlimen dan identiti setempat.
